redis入门篇之环境搭建

发表于 2021-02-20 更新于 2021-06-01 分类于 数据库

搭建环境

  • Ubuntu 20.04
  • redis 6.0.10

安装

  • 第一种安装方式 apt
  • 第二种安装方式 编译安装
    官方下载地址

    执行完 make install 命令后会在当前目录的src目录下生成redis的相关命令，自动会将命令拷贝到 /user/local/bin 目录下

管理

启动

  • 默认
  • 加配置文件方式

停止

  • 通过客户端方式停止

  • 通过杀死进程的方式停止

连接

  • 默认

  • 指定IP与端口号方式

客户端返回值

  • 状态回复

    127.0.0.1:6379> ping

    pong

  • 错误回复

    127.0.0.1:6379> hget hello field

    (error) WRONGTYPE Operation aginst

  • 整数回复

    127.0.0.1:6379> incr hello

    (integer) 1

  • 字符串回复

    127.0.0.1:6379> get hello

    “world”

  • 多行字符串回复

    127.0.0.1:6379> mget hello field

    1) “world”
    2) “bar”
